# Dealer MCP Server

## Descripción
Servidor de Protocolo de Contexto de Modelo (MCP) para gestión de concesionario de vehículos.

## Características
- Gestión de inventario de vehículos
- Calculadora de financiamiento
- Agenda de pruebas de manejo

## Prerequisitos
- Bun 1.0+

## Instalación
```bash
bun install
```

## Ejecución
```bash
# Iniciar servidor
bun run start

# Ejecutar pruebas
bun test

# Construir proyecto
bun run build
```

## Herramientas
- Inventario
- Financiamiento
- Pruebas de Manejo
